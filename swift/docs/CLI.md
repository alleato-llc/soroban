# The `soroban` CLI (Swift)

The `SorobanCLI` executable — the language without the app. One `Calculator` per
invocation; mode by invocation shape (one-shot args / piped stdin / interactive
REPL). Depends on `Anzan` only, and deliberately has no sheet layer.
