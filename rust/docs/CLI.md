# The `soroban` CLI (Rust)

The `soroban` binary — the language without the app. One-shot args / piped stdin
/ interactive REPL, mirroring the Swift CLI's behavior. Depends on the `anzan`
crate only; no sheet layer.

> **Status:** skeleton — full content lands in Phase 2. Its non-interactive modes
> are covered by `cli/tests/smoke.rs`; keep it plumbing-thin.
